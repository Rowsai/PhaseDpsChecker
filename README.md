# Phase DPS Checker

戦闘をフェーズ単位で区切り、パーティ全体とメンバー個別の戦闘情報を確認できるDalamudプラグインです。

![Phase DPS Checker icon](images/icon.png)

## インストール

Dalamudのカスタムプラグインリポジトリへ、次のURLを追加してください。

```text
https://raw.githubusercontent.com/Rowsai/Rowsai-Plugins/main/pluginmaster.json
```

## 機能

- パーティメンバーとペット（オーナーへ合算）の敵向けダメージを集計
- DoT / HoT の周期効果を取得し、付与元アクションへ合算
- 通常計測は、パーティメンバーがターゲット可能な敵へ最初にダメージを与えたイベントからフェーズを開始
- 通常計測は、アンカーとなる敵がターゲット不可になったフレーム、または戦闘終了時にフェーズを終了
- アンカー対象を倒した攻撃またはDoT tickを検知した場合は、そのイベント時刻で即座にフェーズを終了
- 設定画面のコンテンツ情報から「絶妖星乱舞」専用フェーズ判定を選択可能（未選択時は通常計測）
- コンテンツリプレイ動作モードでは、Dark KnightやWhite Mageなどジョブ名で表示されるメンバーも集計
- 全体表示: Phase、開始時間、終了時間、DPS、Crit%、DH%、Crit+DH%、最大ダメージ／アクション、Active%、D / Active%、H / Active%
- 計測中のPhase行を青色で強調表示
- 設定でJOBアイコン付きの現在Phase全体集計オーバーレイを表示
- 個人表示: ウェポンスキル、アビリティ、魔法、回復魔法、回復アビリティ、その他の使用回数・詠唱失敗数・ダメージ・回復量
- `プレイヤーは「アクション」の詠唱を中断した。` 形式のバトルログから、パーティ／リプレイメンバーの魔法詠唱失敗数を集計
- 全滅またはコンテンツクリア時に現在表示をクリアし、全フェーズを戦闘履歴へ自動保存
- 履歴表示タブは履歴番号の昇順で表示し、Phase全体／Phase個別とパーティ全体／メンバー個人を選択
- 選択した戦闘履歴を確認画面から個別に削除
- 全体表は Phase 昇順・DPS 降順を初期値として、各ヘッダのクリックで並び替え
- 履歴のアクション内訳表と被ダメージ表も、各ヘッダのクリックで昇順／降順を切り替え
- 開始時間／終了時間はローカル時刻ではなく、戦闘計測開始からの経過時間で表示
- 被ダメージタブで、被ダメージ0のPhaseを含め、敵の攻撃（オートアタック／DoTを含む）をメンバー別に確認
- 被弾時ステータスは、防御系アビリティのバフと野戦治療の陣・アサイラムなどの設置型回復バフのみ表示
- 戦闘履歴をJSONファイルへ自動保存し、プラグイン再起動後も復元
- 履歴の保存先フォルダーをWindows標準のフォルダー選択画面から変更
- 履歴ファイルが500MBを超えると警告を表示し、1GBを超えると危険メッセージとポップアップで通知
- 概要カード、DPS／Active／被ダメージのバー表示、青基調の配色を採用
- `/phasedps` でウィンドウを開閉
- PvP中は集計を自動停止

## 絶妖星乱舞専用フェーズ

- Phase 1: 戦闘開始から「お前たち、『初めて』じゃないな？」を含むケフカの台詞まで
- Phase 2: 「絶ッ！！再現したな、この私を……！」から敵視リストが空になるまで
- Phase 3: 「ボクチンに不可能はなーい！」から、バトルログに「エクスデスは『メテオ』を中断した。」が表示されるまで
- Phase 4: Phase 3終了後の最初のパーティ攻撃判定から、2回目の「どきどきアルテマ」の詠唱完了まで
- Phase 5: Phase 4終了後、「私は破壊し続けよう！」の台詞表示後にケフカがターゲット可能になった時点から、コンテンツクリア直前の最後のパーティ攻撃まで

## 集計定義

- DPS: フェーズ内の敵向け総ダメージ ÷ フェーズ秒数
- Crit/DH/Crit+DH: 敵向けダメージ効果のヒット数を母数に算出
- 使用回数: ActionEffect 1件を1使用として算出（AoEの対象数では増えません）
- 詠唱失敗数: パーティ／リプレイメンバー名と一致する詠唱中断ログをアクション別に集計（使用回数には加算しません）
- 回復量: パーティメンバーを対象とした回復効果の値を合算（オーバーヒール控除なし）
- Active%: `CooldownGroup` または `AdditionalCooldownGroup` がGCDを示す魔法／ウェポンスキルの占有区間を重複なく合算し、フェーズ時間で除算
- D / Active%: 敵へダメージを与えた攻撃系GCDの占有区間のみをフェーズ時間で除算
- H / Active%: パーティを回復した魔法／ウェポンスキル系GCDの占有区間のみをフェーズ時間で除算

## 注意事項

他メンバーの実スキルスピードは取得できないため、Active%はActionシートの基本リキャスト値を使う近似値です。

このプラグインはDPSパーサーに該当するため、Dalamud公式プラグインリポジトリの受け入れ対象外です。私有カスタムリポジトリでの利用を想定しています。
